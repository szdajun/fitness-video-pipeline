# 夜景健身视频色彩增强工具

## 功能说明

本工具用于提升夜景或光线不足环境下拍摄的健身视频画质，通过亮度、对比度、饱和度、白平衡等多项调整，让视频更清晰、更自然。

### 核心功能

| 功能 | 说明 |
|------|------|
| **自动亮度检测** | 分析视频平均亮度（0-100），自动匹配合适参数 |
| **亮度调整** | 提亮暗部，让健身动作更清晰 |
| **对比度增强** | 1.05-1.20，让轮廓更分明 |
| **白平衡校正** | warmth=-5 抵消室内黄灯暖色偏 |
| **饱和度调整** | +10% 让衣服颜色更鲜艳 |
| **CLAHE 暗部增强** | 自适应直方图均衡，暗部细节提升 |
| **时序平滑** | 减少噪点闪烁，让画面更稳定 |
| **轻度锐化** | 补偿手机拍摄造成的画面软边 |

### 亮度等级划分

| 检测值 | 等级 | 自动 brightness | 自动 contrast |
|--------|------|-----------------|---------------|
| < 15 | 极暗 | 18 | 1.20 |
| 15-25 | 较暗 | 12 | 1.15 |
| 25-35 | 略暗 | 8 | 1.10 |
| 35-55 | 正常 | 5 | 1.05 |
| > 55 | 较亮 | 3 | 1.05 |
| > 65 | 很亮 | 0 | 1.00 |

## 使用方法

### 基本命令

```bash
# 自动模式（推荐）— 自动检测亮度并调整参数
python color_grade_video.py "视频路径" --auto -o "输出路径"

# 手动模式 — 指定具体参数
python color_grade_video.py "视频路径" --brightness 8 --contrast 1.05 -o "输出路径"
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--brightness` | 自动 | 亮度 -100~100，负数变暗 |
| `--contrast` | 自动 | 对比度 0.5~2.0 |
| `--saturation` | 1.10 | 饱和度 0.5~2.0 |
| `--warmth` | -5 | 色温 -50~50，正数偏暖/负数偏冷 |
| `--sharpen` | 0.06 | 锐化强度 0~1 |
| `--temporal-smooth` | 0.2 | 时序平滑 0~1，越大越平滑 |
| `--clahe` | 开启 | CLAHE 暗部增强，默认开启 |
| `--no-clahe` | — | 禁用 CLAHE |
| `--auto` | — | 启用自动亮度检测和参数调整 |

### 完整示例

```bash
# 自动处理
python color_grade_video.py "C:/Users/18091/Desktop/视频.mp4" --auto -o "C:/Users/18091/Desktop/视频_增强.mp4"

# 手动微调（适用于已经看过自动效果，想进一步调整）
python color_grade_video.py "视频.mp4" --brightness 10 --contrast 1.10 --saturation 1.15 --warmth -3 -o "输出.mp4"

# 关闭所有增强，仅保留基础处理
python color_grade_video.py "视频.mp4" --brightness 0 --contrast 1.0 --saturation 1.0 --no-clahe -o "输出.mp4"
```

## 保留原始音轨

处理后的视频默认无音频（OpenCV 读取限制）。如果原视频有音轨，用 FFmpeg 合并：

```bash
ffmpeg -y -i 增强视频.mp4 -i 原视频.mp4 -c:v libx264 -crf 23 -preset fast -c:a aac -b:a 128k -shortest 输出_final.mp4
```

**参数说明：**
- `-crf 23` — 画质平衡（值越小越清晰，18-23 之间最佳）
- `-preset fast` — 编码速度
- `-b:a 128k` — 音频码率
- `-shortest` — 以短的视频为准自动截断

## 工作流程

```
原始视频 → 自动亮度检测 → 色彩参数调整 → 输出增强视频 → (可选)合并音轨 → 最终视频
```

## 适用场景

- 晚上健身房内自拍，光线偏暗偏黄
- 白天室内但灯光较暗
- 视频整体偏灰、对比度不足
- 想要让健身服颜色更鲜艳

## 不适用场景

- 已经很亮的视频（会自动跳过或用保守参数）
- 需要美颜、瘦脸等人物变形处理（请用 `sexy` 预设）
- 需要横屏转竖屏短视频（请用完整 pipeline）
