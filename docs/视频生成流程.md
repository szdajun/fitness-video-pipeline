## 总体架构

当前 pipeline 入口是 `main.py`，通过 stage 顺序处理视频：

```
输入视频 (横版 16:9)
    ↓
[01] pose_detect        — YOLO 姿态检测，保存关键点
[02] stabilize          — 视频稳定化（vidstab）
[03] h2v_convert        — 横转竖（9:16），领操人追踪
[04] ken_burns          — Ken Burns 运镜（dual/smooth 模式）
[05] body_warp          — TPS 体型调整（瘦身/长腿）
[06] color_grade        — 调色（亮度/对比度/饱和度/暖色）
[07] export             — 最终合成输出
    ↓
输出视频 (横版 16:9)
```

完整 stage 列表（按执行顺序）：

| # | Stage | 说明 |
|---|-------|------|
| 01 | pose_detect | YOLO 姿态检测，输出关键点 |

## 使用方法

### 单视频处理

```bash
cd F:/wkspace/fitness-video-pipeline
python main.py process "C:/Users/18091/Desktop/短视频素材/丽丽6.mp4" --preset sexy
```

### 批量处理

```bash
python main.py batch -i "C:/Users/18091/Desktop/短视频素材" -o "output"
```

### 常用参数

```bash
--preset sexy       # 使用 sexy 预设
--preview          # 预览模式（前3秒）
--no-stabilize     # 禁用稳定化
--no-body-warp     # 禁用体型调整
--crf 20           # 输出质量（18-28，越小越大）
--full-video       # 生成完整视频（跳过精华片段选取）
```

### 可用预设

- `natural` — 自然风格
- `dramatic` — 戏剧化
- `clean` — 干净风格
- `sexy` — 性感风格（默认推荐）
- `night_gym` — 夜店健身
- `gimbal` — 云台稳定风格
- `beauty` — 美颜风格
- `youtube` — YouTube 发布
- `shorts` — 短视频

## 配置文件

Preset 配置文件位于 `presets/` 目录，YAML 格式：

```yaml
# presets/sexy.yaml 示例
ken_burns:
  mode: dual
  dual_close_zoom: 1.2
  dual_cycle_seconds: 8

output:
  width: 1920
  height: 1080
  crf: 22
  audio_bitrate: 128k

stages:
  stabilize: false
  ken_burns: true
  skin_smooth: true
  face_beautify: true
```
