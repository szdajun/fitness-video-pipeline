# 健身视频生成流程

## 完整流程

```
健身群下载视频
    ↓
放入素材目录: C:\Users\18091\Desktop\短视频素材
    ↓
Step 1: python fitness_processor.py --once
    - 自动亮度检测
    - 色彩增强（提亮/降噪/肤色保护）
    - 保留原音
    - 输出到: F:\wkspace\fitness-video-pipeline\output\YYYY-MM-DD\
    ↓
Step 2 (可选): python loop_video.py --auto
    - 时长不足5分钟的视频自动循环补足
    ↓
上传 YouTube
```

## Step 1: 素材处理

```bash
# 单次处理素材目录中的所有新视频
python fitness_processor.py --once

# 查看处理进度
```

**处理效果：**
- 自动亮度检测 + Gamma校正
- 肤色保护（主体人物清晰，后排模糊）
- 降噪
- 白平衡校正（抵消黄灯暖色）
- H.265 编码压缩

**配置文件：** `fitness_processor_config.yaml`
```yaml
brightness: 5        # 自动亮度参数
contrast: 1.05
saturation: 1.10
warmth: -5           # 抵消黄灯
gamma: 1.1           # Gamma校正
denoise_h: 3         # 降噪
skin_protect: true   # 肤色保护
skin_blend_strength: 0.2
temporal_smooth: 0.2
clahe: true
encoder: libx265     # H.265编码
crf: 23
```

## Step 2: 视频循环（可选）

```bash
# 自动循环所有不足5分钟的视频
python loop_video.py --auto

# 手动指定视频循环N次
python loop_video.py "视频.mp4" -o "输出.mp4" --count 3
```
