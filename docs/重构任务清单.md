# 健身视频优化流水线重构任务清单

## 使用说明

这份清单用于配合 [重构设计方案.md](/F:/wkspace/fitness-video-pipeline/docs/重构设计方案.md) 执行。设计方案偏“为什么这样做”，本清单偏“接下来具体做什么”。

建议使用方式：

1. 按 PR 顺序推进
2. 每完成一项就勾选
3. 每个 PR 完成后跑一次最小回归
4. 不要跨 PR 大范围混改
