# 夜景健身视频高光优化算法方案

## 文档目的

这份文档用于沉淀针对夜景广场健身视频的高光优化方案，目标是解决以下问题：

- 顶灯或侧上方灯源过曝
- 白衣、浅色衣物发飘
- 地面反光区域刺眼
- 整体提亮后高光进一步炸掉

适用场景：

- 夜景
- 棚下/广场
- 多人健身
- 领操人穿白衣或浅色衣物
- 原视频码率不高，整体不适合暴力锐化或暴力提亮

---

## 总体思路

不建议用“整体降亮度”解决高光问题。  
更合理的方案是三层处理：

1. 基础高光压缩
2. 白衣/低饱和亮区保护
3. 大面积灯区与灯光溢出保护

核心原则：

- 只压过亮区域
- 不压正常中间调
- 尽量平滑过渡
- 避免出现脏边界和局部突变

---

## 第一层：基础高光压缩

## 目标

压制亮度通道中过亮的部分，例如：

- 顶灯
- 亮地面
- 白衣高光
- 亮柱面反光

## 建议色彩空间

优先使用 `LAB` 的 `L` 通道，而不是直接在 BGR 上改。

原因：

- `L` 更接近亮度感知
- 不容易直接破坏色相
- 比单纯改 RGB 通道更自然

## 算法

设：

- `L` 为亮度通道，范围 `0~255`
- `threshold` 为高光开始压制阈值
- `strength` 为压缩强度

当 `L <= threshold` 时不处理。  
当 `L > threshold` 时，对超出部分做平滑 roll-off：

```text
excess = L - threshold
compressed = threshold + excess / (1 + strength * excess / range)
```

这比硬截断更自然。

## 参考实现

```python
import cv2
import numpy as np


def protect_highlights(frame, strength=0.35, threshold=185, blur_ksize=5):
    """
    基础高光保护。
    strength: 0~1，越大压得越明显
    threshold: 从该亮度开始压高光
    blur_ksize: mask 平滑核，建议奇数
    """
    if strength <= 0:
        return frame

    lab = cv2.cvtColor(frame, cv2.COLOR_BGR2LAB).astype(np.float32)
    l = lab[:, :, 0]

    excess = np.maximum(l - threshold, 0.0)
    if np.max(excess) < 1:
        return frame

    compressed = threshold + excess / (
        1.0 + strength * excess / max(1.0, 255.0 - threshold)
    )

    mask = np.clip(excess / max(1.0, 255.0 - threshold), 0.0, 1.0)
    if blur_ksize >= 3 and blur_ksize % 2 == 1:
        mask = cv2.GaussianBlur(mask, (blur_ksize, blur_ksize), 0)

    l_new = l * (1.0 - mask) + compressed * mask
    lab[:, :, 0] = np.clip(l_new, 0, 255)

    return cv2.cvtColor(lab.astype(np.uint8), cv2.COLOR_LAB2BGR)
```

## 第二层：白衣/低饱和亮区保护

## 为什么需要这一层

夜景健身视频中，领操人常穿白衣或浅色上衣。  
即使整体高光压缩做了，白衣仍可能出现：

- 纹理发飘
- 局部细节消失
- 胸口和肩膀区域亮成一片

这类区域的特点是：

- 亮度高
- 饱和度低
- 多数不是灯源，而是衣服反光

因此可以在 `HSV` 或 `LAB` 上单独识别“高亮低饱和区域”。

## 算法思路

1. 转 `HSV`
2. 识别：
   - `V` 高
   - `S` 低
3. 对这类区域做轻微亮度压缩
4. 只压一点，避免白衣发灰

## 参考实现

```python
def protect_bright_neutral_regions(
    frame,
    strength=0.20,
    value_threshold=200,
    sat_threshold=60,
    blur_ksize=5,
):
    """
    保护白衣、浅灰衣物、浅色高反光区域。
    只针对高亮且低饱和区域。
    """
    if strength <= 0:
        return frame

    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV).astype(np.float32)
    h, s, v = cv2.split(hsv)

    bright_mask = np.clip((v - value_threshold) / max(1.0, 255.0 - value_threshold), 0.0, 1.0)
    low_sat_mask = np.clip((sat_threshold - s) / max(1.0, sat_threshold), 0.0, 1.0)
    mask = bright_mask * low_sat_mask

    if blur_ksize >= 3 and blur_ksize % 2 == 1:
        mask = cv2.GaussianBlur(mask, (blur_ksize, blur_ksize), 0)

    # 只轻微压缩，避免白衣发灰
    v_new = v * (1.0 - strength * mask)
    hsv[:, :, 2] = np.clip(v_new, 0, 255)

    return cv2.cvtColor(hsv.astype(np.uint8), cv2.COLOR_HSV2BGR)
```

## 推荐参数

```yaml

```

## 说明

这一层不要压得太重。  
目标不是“让白衣变灰”，而是“让白衣恢复一点层次”。

## 第三层：大面积灯区与灯光溢出保护

## 为什么需要这一层

夜景广场健身视频里，真正最刺眼的通常不是人物，而是：

- 右上角大灯
- 顶部棚灯
- 灯柱附近大片泛白区域

这些区域特点是：

- 很亮
- 连续面积大
- 往往靠近边缘或顶部
- 对局部亮度压缩的容忍度更高

所以可以单独识别“大面积超亮区域”，对其做额外抑制。

## 算法思路

1. 在亮度通道上找超亮区域
2. 做连通域分析
3. 只保留面积较大的亮斑
4. 对这些区域额外压一点亮度
5. 平滑融合，避免产生难看边缘

## 参考实现

```python
def suppress_large_light_regions(
    frame,
    strength=0.18,
    threshold=235,
    min_area=2500,
    blur_ksize=21,
):
    """
    额外压制大面积灯区和灯光溢出区域。
    """
    if strength <= 0:
        return frame

    lab = cv2.cvtColor(frame, cv2.COLOR_BGR2LAB).astype(np.float32)
    l = lab[:, :, 0]

    binary = (l >= threshold).astype(np.uint8)
    num_labels, labels, stats, _ = cv2.connectedComponentsWithStats(binary, connectivity=8)

    mask = np.zeros_like(l, dtype=np.float32)
    for i in range(1, num_labels):
        area = stats[i, cv2.CC_STAT_AREA]
        if area >= min_area:
            mask[labels == i] = 1.0

    if np.max(mask) < 0.5:
        return frame

    if blur_ksize >= 3 and blur_ksize % 2 == 1:
        mask = cv2.GaussianBlur(mask, (blur_ksize, blur_ksize), 0)
        mask = np.clip(mask, 0.0, 1.0)

    l_new = l * (1.0 - strength * mask)
    lab[:, :, 0] = np.clip(l_new, 0, 255)

    return cv2.cvtColor(lab.astype(np.uint8), cv2.COLOR_LAB2BGR)
```

## 三层组合建议

推荐顺序：

1. 基础高光压缩
2. 白衣/低饱和亮区保护
3. 大面积灯区保护

建议封装为一个统一入口：

```python
def optimize_night_highlights(frame, cfg):
```

## 调参建议

### 如果灯区还是太炸

- 提高 `highlight_protect`
- 降低 `highlight_threshold`
- 提高 `light_region_protect`

### 如果白衣发灰

- 降低 `white_protect`
- 提高 `white_value_threshold`

### 如果人物整体暗了

- 不要先降全局 brightness
- 先降低灯区保护强度
- 或提高 shadow / midtone 提升

### 如果边缘出现脏感

- 提高 blur 参数
- 不要用太硬的二值 mask

---

## 推荐实施顺序

最推荐的顺序是：

1. 先接基础高光压缩
2. 再接白衣保护
3. 最后接大面积灯区保护

原因：

- 第一层收益最大
- 第二层对当前样本（白衣领操人）非常有价值
- 第三层是加分项，但实现稍复杂

---

## 验收重点

实现后建议重点观察：

1. 右上灯区是否更收敛
2. 白衣胸口、肩部是否恢复层次
3. 地面高反光是否更柔和
4. 人物主体是否没有被整体压暗
5. 画面是否没有出现脏边界或局部发灰

---

## 总结

针对这类夜景健身视频，最合适的高光优化不是“整体变暗”，而是：

1. 对高亮区域做平滑 roll-off
2. 对白衣等低饱和亮区做单独保护
3. 对大面积灯区做额外抑制

这样可以在尽量不伤害人物主体亮度的前提下，减少夜景灯光过曝和白衣发飘问题，明显提升成片自然度与耐看度。
